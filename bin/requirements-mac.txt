# Nearcade macOS Requirements
#
# MANDATORY:
#   - Node.js (v18+): https://nodejs.org/
#   - Python 3.8+: https://www.python.org/
#   - npm: comes with Node.js
#
# FOR KBM SUPPORT (Keyboard/Mouse):
#   - pyautogui Python package: installed by pip (see below)
#
# FOR GAMEPAD SUPPORT (Experimental - KBM Emulation):
#   - pynput Python package: translates gamepad input to keyboard/mouse
#   - Requires Accessibility permission (System Settings → Security & Privacy)
#   - Supports gamepad through emulated keyboard/mouse events
#   - Works with Steam Input for remapping
#
# SETUP INSTRUCTIONS:
#   1. Install Node.js and Python 3 (e.g., via Homebrew or direct download)
#      brew install node python@3.11
#   2. Extract Nearcade to a folder
#   3. Open Terminal in the Nearcade folder
#   4. Run: pip3 install -r bin/requirements-mac.txt
#   5. Run: npm install
#   6. Grant Accessibility permission:
#      → System Settings → Security & Privacy → Accessibility
#      → Add the Nearcade app to the list
#   7. Run: ./bin/start.cmd  OR  npm run electron
#
# KNOWN LIMITATIONS:
#   - No direct gamepad injection (macOS system limitation)
#   - Gamepad input is emulated via keyboard/mouse events (works with games)
#   - Motion controls not supported
#   - Audio playback uses native afplay command
#
# Python packages
pyautogui
pynput
