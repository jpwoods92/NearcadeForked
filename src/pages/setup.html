Welcome to Nearcade

This quick setup will get your system ready for hosting and playing local co-op games over the internet. You'll set up system dependencies, choose your tunnel provider, and explore optional modules.

Nearcade is an open-source platform that lets you play local co-op games over the internet with friends. It uses secure tunnel technology so your friends can connect directly to your game session from anywhere in the world, no port forwarding or complex network setup needed.

System Setup

Nearcade needs system dependencies (drivers and virtual controllers) to run properly. Choose how you'd like to proceed.

Linux

Run the terminal setup script to install udev rules, PipeWire routing, and uinput drivers.

Windows

Run the PowerShell setup script as Administrator to install ViGEmBus and configure audio.

macOS

Manual steps are available in the documentation. No automated setup script exists yet.

Advanced Modules

      
Nearcade supports optional modules and integrations. Configure these later from the Host dashboard.

      

        

          

            
            

              
VR Streaming

              
WiVRn integration for wireless VR headset streaming.

            

          

          

            
            

              
Discord RPC

              
Rich Presence with join secret for one-click invites.

            

          

          

            
            

              
Arcade Directory

              
Public session listing for global discovery.

            

          

          

            
            

              
Viewer Accounts

              
Optional session passwords for private games.

            

          

          

            
            

              
Multi-Controller

              
Up to 4 controllers with individual gamepad mapping.

            

          

          

            
            

              
Voice Chat

              
Built-in viewer voice with per-viewer volume control.
