Controller Settings

Virtual controller emulation options. Changes apply to all connected viewers immediately.

Disable Xbox 360 emulation
Forces Xbox One S profile for all Xbox controllers instead of the 360-compat fallback. Use this if games are showing wrong button prompts.
⚠ Existing controllers will be re-enumerated on next connect. Reconnect viewers after toggling.
Native PlayStation Profiles
Allows DualShock 4 and DualSense controllers to appear natively to games (supports native lightbar/touchpad if the game allows it). If disabled, they will automatically emulate Xbox controllers for maximum compatibility.
Enable Motion Controls (Gyro)
Passes gyroscope and accelerometer data to the host.
Note: Viewers must use a Chromium-based browser and manually grant WebHID permissions.
Default Input Method for Viewers
Set the default input mode for new viewers. They can still be switched individually from the roster.
