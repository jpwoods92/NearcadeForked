# Nearsec を一緒に始めましょう

Nearcade を使用すると、WebRTC を使用してインターネット経由でローカル ゲームを友人と共有できます。

## ホスティング オプション
セッションをホストするには 2 つの方法があります。

1. プライベート トンネル: Cloudflare または Zrok を通じてカスタム トンネルをセットアップし、友達用の永久リンクを作成できます。これはプライベート グループに最適です。
2. Nearsec Arcade: The Arcade は、ローカルの協力ゲームを見つけるためのパブリック ディレクトリです。ロビーをアクティブに保つため、セッションは 80 分に制限されています。セッションをリストするには、Cloudflared や Zrok などの検証済みのトンネリングプロバイダーを使用する必要があります。 https://nearcade.cutefame.net/arcade でパブリック ロビーを表示し、アクティブなゲームに参加できます。

## セッションの開始
ホスティングを開始するには、次の手順に従ってください。

1. Node.js バージョン 18 以降と Python 3 をマシンにインストールします。
2. ほとんどのユーザーは、コンパイルされた実行可能ファイルを直接起動します。アプリは権限とトンネルを自動的に管理します。
3. ソース コードを使用する場合は、ターミナルを開いて bin フォルダーに移動し、セットアップ スクリプトを実行します。

    ```bash
    cd bin
    sudo ./linux_setup.sh
    ```

4. Linux アプリケーションは、uinput カーネル モジュールをロードする許可を要求します。この手順は、ネイティブ仮想コントローラーを構築するために必要です。
5. [ホスト セッション] ボタンをクリックして、キャプチャ ダッシュボードを開きます。
6. 生成されたリンクとセッション PIN を視聴者に送信します。 Rust ルーターは、ホスト アプリケーションがビューアからの PIN を検証するまで、すべてのビデオおよびオーディオ ストリームをブロックします。

このプロジェクトでは、コード生成と構造計画に人工知能の大規模言語モデルを使用します。
