# ホストの使用法とダッシュボードのガイド

ホスト ダッシュボードは、ストリーム、ビューア、システム オーディオを管理するためのコントロール センターです。

## ビデオとオーディオのキャプチャ
セッションを開始すると、Nearsec は Wayland、X11、Windows Graphics Capture などのネイティブ オペレーティング システム API に接続します。
* Linux のオーディオ ルーティング: Nearsec は、NearsecVirtualCapture 仮想シンクを自動的に作成します。システムは、正確な PipeWire ノード プロパティを使用して、ゲーム オーディオをこのシンクに自動的にルーティングします。これにより、個人のデスクトップ オーディオや音声チャットがストリームから遮断されます。
* 音量コントロール: 聴覚を保護するために、仮想シンクの音量は自動的に 70% に制限されます。

## プレーヤー名簿と入力権限
視聴者は参加すると名簿に表示されます。入力モードを完全に制御できます。
* ゲームパッド: ネイティブ仮想コントローラーを作成します。
* Raw キーボードとマウス: 直接入力パススルー。
* エミュレートされたキーボードとマウス: キーボード入力を仮想ゲームパッドにマッピングします。これは、レトロ ゲームや格闘ゲームにネイティブ キーボード サポートがない場合に役立ちます。
* スロットをロック: 南京錠のアイコンをクリックすると、ランダムな視聴者がアクティブなプレイヤーのスロットを乗っ取るのを防ぎます。

## ボイスチャット管理
視聴者はマイク音声をホストに直接送信できます。
* 赤いマイクアイコン: ローカルでミュートされています。音声は聞こえませんが、音声はサーバーに到着しています。
* 灰色のマイクアイコン: 強制的にミュートされます。サーバーは、すべてのユーザーの帯域幅を節約するために、オーディオ パケットを完全にドロップします。

このプロジェクトでは、コード生成と構造計画に人工知能の大規模言語モデルを使用します。
